# Solana 持币分红代币创建教程

# 📌 核心摘要

功能定位：Solana 持币分红代币创建教程。基于 Token 2022 标准，通过可视化界面实现“交易即分红”的代币模型，无需编写智能合约。

核心机制：

自定义交易税率：在创建时设定转账手续费率。

自动分发：系统自动将收取的手续费按比例分配给持币者，实现被动收益。

前置硬性条件：

钱包准备：必须安装 Solana 生态钱包（如 Phantom）。

资金门槛：钱包内需至少持有 0.07 SOL 用于支付创建费用及 Gas。

网络环境：需配置网络代理工具以确保连接 Solana 主网的稳定性。

# 准备事项

1.准备设备，一台电脑或手机

2.Solana 钱包（幻影钱包Phantom安装教程）

3.钱包内最少准备0.08个SOL

4.准备好翻墙软件（VPN/加速器），保证网络通畅

# 创建持币分红代币操作步骤

1.连接幻影钱包Phantom

打开发币链接地址：<a href="https://sol.gtokentool.com/zh-CN/Token/createToken">https://sol.gtokentool.com/zh-CN/Token/createToken</a>

点击连接钱包，并选择 Main 网络节点，这里使用测试网演示。

2.填写代币相关信息并上传LOGO文件

完成钱包连接之后，先选择代币模式，接下来就是填写代币信息，具体如下：

代币名称：代币的名称信息（如GTOKEN），支持英文、中文以及中英文混合，最多32个字符。

代币简称：代币的简称信息（如G），支持英文、中文以及中英文混合，最多10个字符。

代币精度：代币的精度位数，默认为9，精度与你能填写的最大总供应量有关。

总供应量：代币的总供应量，当精度为9时，总供应量不能超过100亿；当精度为8时，总供应量不能超过1000亿，以此类推。

简介(选填）：填写你的代币简介。

Logo：代币头像，可在钱包中显示logo代币图片（上传图片大小最大不超过2M）。

转账费率：每次交易或转账所需的手续费率，单位为百分比。

最高费用：单笔转账或交易时的最大手续费用上限。

权限钱包地址：控制此代币的所有权限，默认为当前链接的钱包地址。

3.添加选填内容

如果你想添加联系方式等，可以点击打开“添加联系方式”按钮，选择需要填写的内容进行填写。

官网：官网链接地址。

电报：电报（Telegram）链接地址。

推特：推特（Twitter）链接地址。

Discord: Discord链接地址。

4. 高级设置填写
   
提示：如果您不了解高级功能设置，请勿输入或修改。

权限委托：将您的代币权限委托给另一个地址，该地址将能够燃烧或转移任意数量的代币。

计息利率%(0.00 - 100)：为您的代币设定利率，模仿债券等现实世界的资产，并随时获得利息。需要先设置交易费率。

不可转账（灵魂绑定代币）：默认关闭，使代币不可转让给其他地址，通常适用于独特成就等类型项目。

5. 高级选项设置
   
如果您想要自定义代币合约，可以使用靓号合约功能创建代币。

代币更新权限、冻结权限、增发权限设置。

放弃更新权限：“放弃更新权限”意味着您将无法修改令牌元数据。

放弃冻结权限：如果您放弃冻结权限，则意味着您将无法冻结持有者钱包中的代币。

放弃增发权限：“放弃增发权限”对于让投资者感到更加安全和作为代币的成功是必要的。如果您放弃铸币权，则意味着您将无法铸造更多代币供应。

6.点击"Confirm"，创建代币

确认信息之后，点击"Confirm"按钮，之后会跳出钱包提示，点击"Confirm"支付费用，即可完成创建。

7.查看代币

创建成功后会弹出一个提示框提示创建成功，并展示你的提现钱包和代币地址。

创建成功后可以前往管理代币，在分红代币列表下可以找到刚才创建的代币，可以进行进一步的Solana发币管理。或者直接在上方的搜索框内输入代币地址进行查找。

或者前往SOL区块链浏览器去查看代币信息。

# ❓常见问题 FAQ

Q: Solana上发币需要合约开源吗？
A: 不需要。Solana上的代币是通过官方发布好的合约创建出来的账户，所有的SPL代币用的都是一个合约。

Q: 为什么发币之后钱包显示未知代币/不显示logo?
A: Phantom钱包有时候信息有延迟，需要一段时间才能显示logo和代币名称等信息。可以在Solana浏览器上搜索代币信息进行查看。

Q: 上传logo需要收费吗？
A: 不需要，平台采用的是IPFS存储方案，免费上传。
